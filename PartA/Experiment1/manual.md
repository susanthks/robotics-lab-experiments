Theory
Arduino UNO
Arduino UNO is based on the ATmega328P microcontroller and provides:
    • 14 Digital I/O Pins
    • 6 Analog Inputs
    • USB Interface
    • UART Communication

Arduino IDE
Arduino IDE is used to:
    • Write code
    • Compile sketches
    • Upload programs to Arduino
LED Interfacing
LEDs are output devices used for indication.
LCD Interfacing
16×2 LCD can display:
    • Text
    • Numbers
    • Sensor Values
Serial Monitor
Used for:
    • Debugging
    • Monitoring sensor readings
